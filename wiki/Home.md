# Prompt Engineering for AI Data Engineers - Wiki

Welcome to the official wiki for the **Prompt Engineering for AI Data Engineers - Multi-Model Edition** course. This comprehensive resource is designed to help data engineers master prompt engineering techniques for AI-powered data workflows.

## 🎯 Course Overview

This course provides practical, hands-on training for data engineers to leverage Large Language Models (LLMs) for real-world data engineering challenges. Unlike generic prompt engineering courses, this curriculum is specifically tailored to address data engineering tasks including data cleaning, ETL/ELT pipeline enhancement, data governance, SQL generation, and real-time data processing.

## 🚀 Getting Started

1. **Clone the repository**: `git clone https://github.com/nellaivijay/prompt-engineering-for-ai-data-engineers.git`
2. **Install dependencies**: `pip install -r requirements.txt`
3. **Configure API keys**: Copy `config/api_keys.example.yaml` to `config/api_keys.yaml` and add your API keys
4. **Start with Module 1**: Begin with the foundational concepts
