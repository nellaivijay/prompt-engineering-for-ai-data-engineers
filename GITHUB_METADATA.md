## Repository Description

**Short Description (280 characters max):**
```
Comprehensive prompt engineering course for AI data engineers with multi-model support (OpenAI, Anthropic, Gemini, Llama, Mistral). Master data cleaning, ETL optimization, SQL generation, and cost-effective AI implementation in data pipelines.
```

**Long Description:**
```
A comprehensive, production-ready course on prompt engineering specifically designed for AI data engineers, featuring multi-model support across major AI platforms including OpenAI, Anthropic, Google Gemini, Meta Llama, Mistral, and more. Master advanced techniques for data cleaning, ETL/ELT optimization, SQL generation, data governance, and cost-effective AI implementation in data pipelines.

This 12-module course provides practical, hands-on training for data engineers to leverage Large Language Models (LLMs) for real-world data engineering challenges. Includes enterprise-grade implementations, modern data stack integration, local deployment options, and systematic model benchmarking.
```
