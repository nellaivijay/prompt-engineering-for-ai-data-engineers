# Module 11: Model-Specific Prompt Engineering

Master prompt engineering techniques specific to each major AI model provider and architecture.

## Learning Objectives

- Apply OpenAI-specific techniques (function calling, structured output)
- Use Anthropic-specific techniques (Constitutional AI, context windows)
- Leverage Gemini-specific techniques (multimodal, long context)
- Implement Llama-specific techniques (fine-tuning, quantization)
- Apply Mistral-specific techniques (function calling, mixtures of experts)
- Optimize open-source models (quantization, distillation)

## Key Topics

- **OpenAI**: GPT-specific features and optimizations
- **Anthropic**: Claude-specific techniques
- **Google**: Gemini-specific capabilities
- **Meta**: Llama-specific methods
- **Mistral**: Mistral-specific features
- **Open Source**: Optimization techniques
